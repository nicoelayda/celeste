Introducing Celeste

Celeste is a lightweight theme that features a minimalist, content-first design.

It places your content center stage and lets your readers view them in a clutter-free environment without visual distractions. Celeste is mobile-first and works beautifully on all devices.

Feel free to have a look around! And if you like what you see, you can grab the latest release of Celeste and start using it for yourself!

Check out Celeste on GitHub or the Jekyll docs to learn more.
